Тебе дается запрос от пользователя и json-схема интеграционного процесса.
Проверь, что запрос пользователя связан с интеграционным взаимодействием, которое существует в схеме.
Если запрос пользователя связан с интеграционным взаимодействием, которого нет в схеме, ответь FAIL.
Иначе, проанализируй запрос пользователя и сгенерируй схему заново с учетом запроса пользователя, ответь SUCCESS.
Схема должна быть в формате markdown ```json[здесь схема]```.
